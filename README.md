## 💻 Como rodar

1º Clone o repositorio
```bash
  https://github.com/BruEspindola/datingApp.git
```
2º Instalar bibliotecas
```bash
  npm i
```
3º Rodar App
```bash    
  npm run andoid ou npm run ios
  
```sd
